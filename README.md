## About

App to know the time based on pre-defined timestamps

## Installation

Follow these steps

```bash
git clone https://github.com/amrfoley/time-app.git
```
```bash
cd time-app
```
```bash
cp .env.example .env
```
insert Database name, user & password
```bash
php artisan migrate
```
```bash
php artisan insert:timetamps
```
visit [app](http://localhost:8000/)
